# Django Job Application Form

A simple web application for collecting job applications with database storage and email confirmation.

## Features
- User-friendly application form
- Database storage for all submissions
- Automatic email confirmation to applicants
- Responsive design for all devices

## How to Use
1. Run development server: `python manage.py runserver`
2. Visit `http://localhost:8000`
3. Submit your application
4. You'll receive an email confirmation and see a success message

## Requirements
- Python 3.11+
- Django 4.0+
